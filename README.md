# AI Coding Journey — From Zero to Product

> A comprehensive AI coding learning guide covering mainstream AI coding tools from beginner to advanced.

🔗 **Live Site**: [https://sjjliqpl.github.io/ai-coding-learning/](https://sjjliqpl.github.io/ai-coding-learning/)

## 📖 Contents

| Page | Description |
|-------------|---------------------|
| **Home** | Core concepts, learning path, tool recommendations, interactive demo |
| **Advanced** | Deep dive into Claude Code, GitHub Copilot, Cursor |
| **MCP** | Model Context Protocol introduction and hands-on |
| **AI Skills** | Agent skill system and creation guide |
| **Terminal** | Terminal basics, core commands, macOS tips, AI tool integration |
| **Markdown** | Markdown syntax quick start and tool recommendations |
| **Git & GitHub** | Version control basics and GitHub collaboration |
| **Models** | Copilot CLI BYOK, Claude Code custom models, Chinese LLM integration guides |

## 🛠️ Tech Stack

- **React 19** + **TypeScript** + **Vite**
- **React Router** (HashRouter) — GitHub Pages compatible
- Dark theme · Glassmorphism cards · Scroll animations
- **i18n** Chinese-English language switching

## 🚀 Development

```bash
npm install       # Install dependencies
npm run dev       # Start dev server (localhost:5173)
npm run build     # Build to docs/
npm run preview   # Preview build
```

## 📁 Project Structure

```
src/
├── components/     # Shared components (Layout, Sidebar)
├── pages/          # Page components
├── styles/         # Global + page styles
├── i18n/           # Internationalization
├── App.tsx         # Route config
└── main.tsx        # Entry
public/
└── screenshots/    # Page screenshots
docs/               # Build output (GitHub Pages)
```
