# ai-coding-learning

这是 "ai-coding-learning" 项目的说明文档，内容基于仓库中的静态 HTML 页面生成并已发布为 GitHub Pages 网站。

项目概览
- 一个面向 AI 开发者的静态文档站点，包含：
  - 首页: 进阶指南（index.html）
  - MCP 协议说明（mcp.html）
  - AI Skills（skills.html）
  - 高级专题（advanced.html）

查看在线站点
- 已为本仓库启用 GitHub Pages（main 分支 /docs 目录），访问：
  https://sjjliqpl.github.io/ai-coding-learning

如何本地运行
1. 在仓库根目录打开：
   git clone https://github.com/sjjliqpl/ai-coding-learning.git
   cd ai-coding-learning
2. 直接在浏览器打开 docs/index.html 即可查看静态页面，或用本地静态服务器（例如：python -m http.server 8000）在本地访问。

贡献指南
- 修改 HTML/CSS 文件后：
  git add <files>
  git commit -m "Update site content"
  git push
